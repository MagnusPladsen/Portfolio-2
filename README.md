## Getting Started

### Installing

1. Clone the repo:

```bash
git clone git@github.com:MagnusPladsen/Portfolio-2.git
```

2. Install the dependencies:

```
npm install
```

### Running

Run application in development mode

```
npm run dev
```

Build application for production

```
npm run build
```
